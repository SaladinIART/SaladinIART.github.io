Featured build

IIoT telemetry stack for machine monitoring and industrial visibility.

This flagship portfolio project simulates a production-ready industrial telemetry pipeline: machine data is collected through industrial protocols, moved through a brokered messaging layer, stored in a time-series database, and visualized for operators and engineers.

Machine simulator Modbus TCP collector MQTT transport Python ingestion service TimescaleDB Grafana dashboards

  • Resilience features including retries, contracts, and migration-ready schemas
