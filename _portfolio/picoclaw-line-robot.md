## Project goal

Build a practical robotics project that demonstrates hardware and software integration while keeping
the implementation approachable, well-documented, and reusable for future learning work.

## What the project demonstrates

- Embedded robotics experimentation with a clean repository structure
- Practical testing notes and iteration mindset instead of demo-only presentation
- Documentation that makes the project understandable to both technical and non-technical readers
- A foundation for broader robotics and ROS2-oriented exploration

## Why it belongs in the portfolio

Although my main direction centers on industrial systems and IIoT infrastructure, this project shows
the adjacent robotics interest that informs my longer-term engineering path. It highlights disciplined
project organization, hardware-software integration, and the ability to communicate implementation choices clearly.
